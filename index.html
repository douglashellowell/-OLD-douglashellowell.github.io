Peppy

A Super Smash Bros Melee scoreboading app for streamers

Peppy is a desktop app built with Electron using NodeJS, HTML and CSS.
Designed with the familiar layout of the character select screen it allows streamers to change elements on a live stream with only a click!

Peppy points to local text and image elements and updates them. Tournament Organisers can pre-load attendee data for quick tag searching.
No more trying to spell M4rioM4ster in a pinch!

Tested at sold-out tournaments i'm proud to say it's been a great success!

Planned features include:

  • Score-updating
  • Player Stats Parsing
  • Auto-score via Slippi
  • Player Profiles
  • ... leave a comment on my github page!
